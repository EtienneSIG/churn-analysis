## Architecture Diagram

The architecture diagram is embedded as a text/ASCII diagram in the main `README.md`. If you want to create a visual version, tools like [Excalidraw](https://excalidraw.com), [draw.io](https://app.diagrams.net), or [Mermaid](https://mermaid.js.org) can be used to produce a PNG/SVG and saved here.

## Usage in README

To reference an asset in the README or other docs:

```markdown
![Lakehouse Tables](assets/02_lakehouse_tables.png)
```

> **Note:** Keep asset file sizes reasonable (< 500KB per image) to keep the repository lightweight. Compress screenshots before committing.
